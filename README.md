# 🎄 Magic Christmas Tree (WebAR)

这是一个基于手势识别的互动圣诞树网页应用。握拳召唤圣诞树，张手让它变回粒子！
Designed by **UP主 zzx**.

## 📦 本地运行 (Quick Start)

如果你想在本地电脑上体验或修改这个项目，请按照以下步骤操作：

### 1. 克隆项目到本地
打开终端 (Terminal) 或 Git Bash，运行：
```bash
git clone [https://github.com/zzxxxjj/christmas-tree.git](https://github.com/zzxxxjj/christmas-tree.git)
cd christmas-tree

```

### 2. 安装依赖

确保你的电脑上已经安装了 [Node.js](https://nodejs.org/) (建议 v16+)。然后运行：

```bash
npm install

```

### 3. 启动开发服务器

依赖安装完成后，运行：

```bash
npm run dev

```

### 4. 打开浏览器体验

启动成功后，终端会显示一个本地访问地址（通常是 `http://localhost:5173`）。
在浏览器中打开该地址，允许摄像头权限，即可开始体验！

> **💡 提示**: 为了获得最佳的手势识别体验，请确保环境光线充足，并让手掌完整出现在摄像头画面中。

## ✨ 功能特点

* **手势识别**: 使用 Google MediaPipe 精准识别手掌与拳头。
* **3D 粒子特效**: 基于 Three.js / React-Three-Fiber 打造的视觉盛宴。
* **节日氛围**: 自带温馨 BGM 和沉浸式圣诞特效。
* **高性能**: 针对不同设备进行了性能优化。

## 🛠️ 技术栈

* React + TypeScript
* Vite
* Three.js / React-Three-Fiber
* Google MediaPipe
* Tailwind CSS
