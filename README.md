# 🎄 Magic Christmas Tree (WebAR)

这是一个基于手势识别的互动圣诞树网页应用。握拳召唤圣诞树，张手让它变回粒子！
Designed by **UP主 zzx**.

## 🎮 在线体验 (Demo)

👉 **点我直接玩 (无需下载):
 https://christmas-tree-git-main-zzxxs-projects.vercel.app/（需要魔法）

## ✨ 功能特点

* **手势识别**: 使用 MediaPipe 识别手掌与拳头。
* **3D 粒子特效**: 基于 Three.js 的视觉盛宴。
* **节日氛围**: 自带温馨 BGM 和圣诞特效。
* **零延迟**: 模型本地化部署，国内秒开。

## 🛠️ 技术栈

* React + TypeScript
* Vite
* Three.js / React-Three-Fiber
* Google MediaPipe
